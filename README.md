# Vorhdam Shadcn Registry

This registry is a collection the components I use daily throughout my work. I decided to make it public so you can add them to your projects too.

<p align="center">
    <img src="/assets/banner.png"/>
</p>

## Components

### Password

**A simple password input component with a visibility toggle.**

Usage guide and more detailed documentation [here](https://github.com/vorhdam/registry/blob/main/public/docs/password.md)

`bunx --bun shadcn@latest add https://vorhdam-registry.vercel.app/r/password.json`

### Flag

**A simple flag component that renders flags as svgs.**

Usage guide and more detailed documentation [here](https://github.com/vorhdam/registry/blob/main/public/docs/flag.md)

`bunx --bun shadcn@latest add https://vorhdam-registry.vercel.app/r/flag.json`

### Phone

**A simple input component that allows you to collect phone numbers.**

Usage guide and more detailed documentation [here](https://github.com/vorhdam/registry/blob/main/public/docs/phone.md)

`bunx --bun shadcn@latest add https://vorhdam-registry.vercel.app/r/phone.json`
